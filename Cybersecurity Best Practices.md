### Core Principles

* Keep systems **updated and patched**
* Use **strong, unique passwords**
* Enable **multi-factor authentication (MFA)**
* **Limit access** to only what is necessary
* Regularly **back up important data**
* Stay alert to **phishing and scams**

---

### Key Practices at a Glance

| Area           | Best Practice                                      | Why It Matters                  |
| -------------- | -------------------------------------------------- | ------------------------------- |
| Passwords      | Use long, unique passwords; use a password manager | Prevents account takeover       |
| Authentication | Enable MFA everywhere possible                     | Blocks unauthorized access      |
| Software       | Install updates promptly                           | Fixes security vulnerabilities  |
| Email & Links  | Verify senders; avoid unknown links/attachments    | Stops phishing attacks          |
| Devices        | Lock screens; encrypt data                         | Protects lost or stolen devices |
| Networks       | Use secure Wi-Fi; avoid public Wi-Fi or use VPN    | Prevents data interception      |
| Backups        | Maintain regular offline/cloud backups             | Allows recovery after attacks   |
| Access Control | Give minimal required access                       | Limits damage if compromised    |
| Monitoring     | Watch for unusual account activity                 | Detects breaches early          |
| Training       | Teach safe online behavior                         | Reduces human error risks       |
